# Options API

## API Reference

Avue-form options configuration API.

### Option Properties

**labelWidth**
- Type: `Number | String`
- Default: `120`
- Description: Label width

**labelPosition**
- Type: `String`
- Default: `'right'`
- Options: `'left' | 'right' | 'top'`
- Description: Label position

**size**
- Type: `String`
- Default: `'medium'`
- Options: `'large' | 'medium' | 'small' | 'mini'`
- Description: Form size

**submitBtn**
- Type: `Boolean`
- Default: `true`
- Description: Show submit button

**emptyBtn**
- Type: `Boolean`
- Default: `true`
- Description: Show reset button

**submitText**
- Type: `String`
- Default: `'提交'`
- Description: Submit button text

**emptyText**
- Type: `String`
- Default: `'重置'`
- Description: Reset button text

**gutter**
- Type: `Number`
- Default: `0`
- Description: Grid gutter

**span**
- Type: `Number`
- Default: `24`
- Description: Default column span

**column**
- Type: `Array`
- Required: `true`
- Description: Column definitions array

**menuBtn**
- Type: `Boolean`
- Default: `false`
- Description: Show menu button

**menuAlign**
- Type: `String`
- Default: `'center'`
- Description: Menu alignment

**See also:** `examples/components/options.md` for detailed examples
