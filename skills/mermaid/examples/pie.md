## Instructions

Pie charts display proportional data, showing how parts relate to a whole.

### Syntax

- Use `pie` keyword
- Title: `title Chart Title` (optional)
- Data format: `"Label" : Value`
- Values should be numeric

### Example

```mermaid
pie title Sales by Product
    "Product A" : 42.5
    "Product B" : 30.2
    "Product C" : 15.8
    "Product D" : 11.5
```
