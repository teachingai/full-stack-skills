# 分享 API

## 概述

分享 API 用于分享内容。

## API 列表

### uni.share

分享内容。

**详细文档**：https://uniapp.dcloud.net.cn/api/share/share.html#share

**参数**：
- `provider` (String) - 分享服务提供商（weixin、sinaweibo、qq）
- `type` (Number) - 分享类型（0-图文、1-纯文字、2-纯图片、3-音乐、4-视频、5-小程序）
- `href` (String) - 分享链接
- `title` (String) - 分享标题
- `summary` (String) - 分享摘要
- `imageUrl` (String) - 分享图片
