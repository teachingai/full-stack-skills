## 什么是离线打包

离线打包是将 uni-app 项目打包成原生 Android 或 iOS 应用的过程。

## 打包流程

### Android 打包

1. 准备 Android 开发环境
2. 配置 manifest.json
3. 使用 Android Studio 打包
4. 签名和发布

### iOS 打包

1. 准备 iOS 开发环境
2. 配置 manifest.json
3. 使用 Xcode 打包
4. 签名和发布
