## Notice boxes

INFO：

```
+------------------------------------------------------------------------------+
| INFO:                                                                        |
| - This is an informational message.                                          |
| - Keep it short and actionable.                                              |
+------------------------------------------------------------------------------+
```

WARN：

```
+------------------------------------------------------------------------------+
| WARN:                                                                        |
| - Do NOT paste tokens here.                                                  |
+------------------------------------------------------------------------------+
```

ERROR：

```
+------------------------------------------------------------------------------+
| ERROR:                                                                       |
| - Operation failed. Check logs and retry.                                    |
+------------------------------------------------------------------------------+
```

SUCCESS：

```
+------------------------------------------------------------------------------+
| SUCCESS:                                                                     |
| - Completed successfully.                                                    |
+------------------------------------------------------------------------------+
```
