### Example: What is Rspack

Rspack is a fast Rust-based web bundler designed to be a drop-in replacement for Webpack, with significantly better performance.

**Key Features**:
- Built with Rust for performance
- Webpack-compatible API
- Fast Hot Module Replacement (HMR)
- Support for most Webpack plugins and loaders

### Example: Why Use Rspack

**Performance Benefits**:
- **Development startup**: 5-10x faster than Webpack
- **Build speed**: 2-5x faster than Webpack
- **HMR speed**: 10x faster than Webpack

**Compatibility**:
- 85% of top 50 Webpack plugins work out of the box
- Most Webpack loaders are compatible
- Similar configuration format

### Example: Comparison with Other Bundlers

**vs Webpack**:
- Faster build times
- Better HMR performance
- Similar API and configuration

**vs Vite**:
- More mature plugin ecosystem
- Better for complex applications
- Supports more Webpack plugins

**vs esbuild**:
- More feature-complete
- Better plugin support
- Production-ready

### Key Points

- Rspack is a high-performance bundler
- Compatible with most Webpack plugins
- Significantly faster than Webpack
- Production-ready (1.0 released)
- Good choice for large projects
