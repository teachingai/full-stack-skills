# Trace View

## Instructions

This example demonstrates trace view for debugging tests.

### Key Concepts

- Trace view features
- Test execution timeline
- Performance analysis
- Debugging with traces

### Example: Using Trace View

```bash
# Run with trace
vitest --trace

# Or in UI mode
vitest --ui
```

### Example: Trace Information

Trace view shows:
- Test execution order
- Time taken per test
- Hook execution
- Async operations
- Performance bottlenecks

### Key Points

- Available in UI mode
- Shows test execution timeline
- Helps identify slow tests
- Useful for debugging
- Performance analysis
