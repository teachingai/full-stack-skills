# uni-app 原生插件开发指南

## 官方文档

参考官方文档：https://nativesupport.dcloud.net.cn/NativePlugin/

## 什么是原生插件

原生插件是用原生代码（Android Java/Kotlin 或 iOS Objective-C/Swift）开发的插件，可以扩展 uni-app 的功能。

## 开发流程

### Android 插件

1. 创建 Android 插件项目
2. 实现插件功能
3. 配置插件信息
4. 打包插件
5. 集成到 uni-app 项目

### iOS 插件

1. 创建 iOS 插件项目
2. 实现插件功能
3. 配置插件信息
4. 打包插件
5. 集成到 uni-app 项目
