### Example: What is Tauri

Tauri is a framework for building desktop applications using web frontend technologies and Rust backend.

### Example: Features

- **Small Bundle Size**: Smaller than Electron
- **Security**: Built-in security features
- **Performance**: Native performance
- **Cross-platform**: Windows, macOS, Linux
- **Web Frontend**: Use any web framework
- **Rust Backend**: Safe and performant

### Example: Architecture

Tauri uses a two-part architecture:
- **Frontend**: Web technologies (HTML, CSS, JavaScript)
- **Backend**: Rust for system operations

### Example: Use Cases

- Desktop applications
- Cross-platform apps
- Native-feeling apps
- Secure applications

### Key Points

- Web frontend + Rust backend
- Small bundle size
- High security
- Cross-platform support
- Native performance
