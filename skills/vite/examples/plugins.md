## Instructions

- Use the official plugins index to select plugins.
- Prefer official plugins first, then community plugins.
- Verify framework compatibility and Vite major version support.

### Example (selection checklist)

```md
- Framework plugin (Vue/React/Svelte)
- JSX/TS support
- Legacy browser support (if needed)
- Testing/build tooling plugins
```

### Example (plugin categories)

```md
- Framework: @vitejs/plugin-vue, @vitejs/plugin-react
- DX: eslint, stylelint, checker
- Build: legacy, visualizer
```

### Notes

- Check plugin maintenance status before adoption.

Reference: https://cn.vitejs.dev/plugins/
