# uni-app 插件市场使用指南

## 官方文档

参考官方文档：https://ext.dcloud.net.cn/

## 什么是插件市场

uni-app 插件市场提供丰富的插件资源，包括组件、API、模板等。

## 插件类型

- 组件插件：UI 组件
- API 插件：功能 API
- 模板插件：页面模板
- 项目模板：完整项目模板

## 安装方式

### HBuilderX 安装

1. 打开插件市场
2. 选择插件
3. 点击导入插件
4. 在项目中使用

### npm 安装

```bash
npm install plugin-name
```
