# Line Chart

## 折线图

折线图用于显示数据随时间或其他有序类别的变化趋势。

### 基础折线图

```vue
<template>
  <view class="chart-container">
    <l-echart ref="chart" :option="option" :echarts="echarts"></l-echart>
  </view>
</template>

<script>
import * as echarts from 'echarts'
import lEchart from '@/uni_modules/lime-echart/components/l-echart/l-echart.vue'

export default {
  components: {
    lEchart
  },
  data() {
    return {
      echarts,
      option: {
        xAxis: {
          type: 'category',
          data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
        },
        yAxis: {
          type: 'value'
        },
        series: [{
          data: [120, 132, 101, 134, 90, 230, 210],
          type: 'line'
        }]
      }
    }
  }
}
</script>
```

### 时间轴折线图

```vue
<template>
  <view class="chart-container">
    <l-echart ref="chart" :option="option" :echarts="echarts"></l-echart>
  </view>
</template>

<script>
import * as echarts from 'echarts'
import lEchart from '@/uni_modules/lime-echart/components/l-echart/l-echart.vue'

export default {
  components: {
    lEchart
  },
  data() {
    return {
      echarts,
      option: {
        tooltip: {
          trigger: 'axis',
          axisPointer: {
            type: 'cross'
          }
        },
        xAxis: {
          type: 'time',
          boundaryGap: false
        },
        yAxis: {
          type: 'value',
          boundaryGap: [0, '100%']
        },
        series: [{
          data: [
            ['2024-01-01', 120],
            ['2024-01-02', 132],
            ['2024-01-03', 101],
            ['2024-01-04', 134],
            ['2024-01-05', 90],
            ['2024-01-06', 230],
            ['2024-01-07', 210]
          ],
          type: 'line',
          smooth: true,
          areaStyle: {}
        }]
      }
    }
  }
}
</script>
```

## 配置说明

### 常用配置项

- `smooth`: 是否平滑曲线显示
- `areaStyle`: 区域填充样式
- `step`: 阶梯折线图类型（'start', 'middle', 'end'）
- `markPoint`: 标记点配置
- `markLine`: 标记线配置
- `markArea`: 标记区域配置
- `symbol`: 标记的图形
- `symbolSize`: 标记的大小

### 样式配置

```javascript
series: [{
  type: 'line',
  lineStyle: {
    color: '#409EFF',
    width: 2,
    type: 'solid' // 'solid', 'dashed', 'dotted'
  },
  itemStyle: {
    color: '#409EFF'
  },
  areaStyle: {
    color: {
      type: 'linear',
      x: 0,
      y: 0,
      x2: 0,
      y2: 1,
      colorStops: [{
        offset: 0, color: 'rgba(64, 158, 255, 0.3)'
      }, {
        offset: 1, color: 'rgba(64, 158, 255, 0.1)'
      }]
    }
  }
}]
```
